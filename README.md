# LDA
LDA algorithm, converts raw scraper data into term-document matrix

# Installation

## Third-Party

```
pip install git+https://github.com/BIDS-projects/lda.git
```

## Development

1. Setup a virtual environment.
2. `pip install -r requirements.txt`.

##Test

```
py.test tests
```

# Usage

```
import bidslda
```
